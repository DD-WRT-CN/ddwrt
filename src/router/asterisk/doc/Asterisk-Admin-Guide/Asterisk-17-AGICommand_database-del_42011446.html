DATABASE DEL

Synopsis

Removes database key/value

Description

Deletes an entry in the Asterisk database for a given family and key.

Returns 1 if successful, 0 otherwise.

Syntax

DATABASE DEL FAMILY KEY
Arguments
  • family
  • key
