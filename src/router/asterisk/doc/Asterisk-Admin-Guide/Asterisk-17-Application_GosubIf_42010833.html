GosubIf()

Synopsis

Conditionally jump to label, saving return address.

Description

If the condition is true, then jump to labeliftrue. If false, jumps to labeliffalse, if specified. In either case, a jump saves the return point in the dialplan, to be returned to with a Return.

Syntax

GosubIf(condition?[labeliftrue:[labeliffalse]])
Arguments
  • condition
  • destination
    • labeliftrue( params ) - Continue at labeliftrue if the condition is true. Takes the form similar to Goto() of [[context,]extension,]priority.
      • arg1[arg1...]
      • argN
    • labeliffalse( params ) - Continue at labeliffalse if the condition is false. Takes the form similar to Goto() of [[context,]extension,]priority.
      • arg1[arg1...]
      • argN
