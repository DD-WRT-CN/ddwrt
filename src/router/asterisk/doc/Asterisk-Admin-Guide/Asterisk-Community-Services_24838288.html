Asterisk Project : Asterisk Community Services

Created by David M. Lee on May 22, 2013

There are several services provided for the development of Asterisk.

  • Issue Tracking
  • Wiki
  • Code Review
  • Version Control
  • Source Browsing
  • Mailing Lists
There is a regular maintenance window every Monday from 9:00 PM to 10:00 PM Central Time, during which services may have intermittent availability while we apply patches, upgrades, etc. If maintenance is required outside of this window, notice will be sent to the asterisk-announce mailing list.
