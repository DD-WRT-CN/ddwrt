VERBOSE

Synopsis

Logs a message to the asterisk verbose log.

Description

Sends message to the console via verbose message system. level is the verbose level (1-4). Always returns 1

Syntax

VERBOSE MESSAGE LEVEL
Arguments
  • message
  • level
