nProbe is a network probe that emits traffic reports in Cisco NetFlow v5/v9 and nFlow (http://www.nflow.org) format. It is based on pcap and it has been written in a portable way in order to virtually run on every Unix platform.

1. This package will be installed under /usr/local/nprobe
2. In order to see all the nProbe options please type /usr/local/nprobe/bin/nprobe -h
3. As nProbe will access the network adapter in promiscuous mode it is necessary to start it as super-user
4. Please fetch the nProbe manual from http://www.ntop.org/nProbe.html
